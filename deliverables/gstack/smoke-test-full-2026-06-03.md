## 2. 发现的问题

### 🟡 P2 - 注册页面表单提交问题

**现象**：通过 Playwright 自动化填写注册表单并点击提交后，页面停留在 `/register`，未跳转到 `/problems`。

**影响**：仅影响自动化测试，不影响实际用户使用（手动注册正常）。

**根因分析**：
- antd Form 的 `onFinish` 回调未被触发
- 可能是 Playwright 的 `fill()` 方法未触发 antd 的 onChange 事件
- 注册 API 本身正常（curl 测试通过）

**建议**：
- 考虑在注册页面添加 API 错误提示
- 或者使用 Playwright 的 `type()` 方法替代 `fill()`
