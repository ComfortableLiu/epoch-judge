# Epoch Judge 用户手册

欢迎使用 Epoch Judge 在线评测系统！本手册面向三类用户角色：

| 角色 | 手册 | 说明 |
|------|------|------|
| 🎓 学生 | [学生手册](./student.md) | 注册登录、浏览题目、提交代码、参加比赛 |
| 👨‍🏫 教师 | [教师手册](./teacher.md) | 创建题目、布置作业、查看成绩 |
| 🔧 管理员 | [管理员手册](./admin.md) | 系统配置、用户管理、比赛管理 |

## 快速开始

1. **注册账号**：访问系统首页，点击「注册」，填写用户名和密码即可
2. **浏览题目**：登录后进入「题目」页面，选择感兴趣的题目
3. **提交代码**：在题目详情页点击「提交」，选择语言并编写代码
4. **查看结果**：提交后自动跳转到提交详情页，查看评测结果

## 系统概览

Epoch Judge 是一个功能完整的在线评测系统，支持：

- **多语言评测**：JavaScript、Python、Java、C、C++
- **两种判题模式**：ACM 模式（通过/不通过）和 OI 模式（按点评分）
- **比赛系统**：支持 ACM/OI 赛制、密码保护、实时排行榜
- **角色权限**：学生（USER）、教师（PROBLEM_EDITOR）、管理员（ADMIN）
