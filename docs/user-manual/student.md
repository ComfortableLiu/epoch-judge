# 学生手册

本手册帮助学生快速上手 Epoch Judge 在线评测系统。

## 1. 注册账号

### 操作步骤

1. 打开系统首页，点击右上角「注册」按钮
2. 填写注册信息：
   - **用户名**（必填）：用于登录的唯一标识
   - **邮箱**（选填）：用于找回密码
   - **密码**（必填）：至少 6 位字符
3. 点击「注册」按钮完成注册

> 💡 注册成功后自动登录，无需再次输入密码。

## 3. 浏览题目

### 操作步骤

1. 登录后点击导航栏「题目」进入题目列表
2. 使用筛选功能查找题目：
   - **关键词搜索**：输入题目标题关键词
   - **标签筛选**：按题目分类标签筛选
   - **难度范围**：拖动滑块设置难度区间（0-5000）
3. 点击题目标题进入题目详情

### 题目列表说明

| 列 | 说明 |
|----|------|
| 编号 | 题目的唯一编号 |
| 标题 | 题目名称，点击可查看详情 |
| 难度 | 题目难度分数 |
| 标签 | 题目分类标签 |
| 状态 | 你的通过状态（已通过/未通过/未尝试） |

## 4. 提交代码

### 操作步骤

1. 在题目详情页点击「提交」按钮
2. 选择编程语言：
   - JavaScript
   - Python
   - Java
   - C
   - C++
3. 选择判题模式：
   - **ACM 模式**：每个测试点通过/不通过，全部通过即 AC
   - **OI 模式**：按点评分，根据通过的测试点数量给分
4. 在代码编辑器中编写代码
5. 点击「提交」按钮

> 💡 可在「设置」页面配置默认编程语言和判题模式，提交时会自动填充。

### 代码编写提示

- 从标准输入读取数据（如 `stdin`、`cin`、`Scanner` 等）
- 将结果输出到标准输出（如 `stdout`、`cout`、`System.out` 等）
- 注意时间限制和内存限制

## 5. 查看提交结果

### 操作步骤

1. 提交后自动跳转到提交详情页
2. 查看评测状态和各测试点结果

### 评测状态说明

| 状态 | 含义 |
|------|------|
| ✅ Accepted (AC) | 答案正确 |
| ❌ Wrong Answer (WA) | 答案错误 |
| ⏱️ Time Limit Exceeded (TLE) | 超出时间限制 |
| 💾 Memory Limit Exceeded (MLE) | 超出内存限制 |
| 💥 Runtime Error (RE) | 运行时错误 |
| 🔧 Compile Error (CE) | 编译错误 |
| ⏳ Pending | 等待评测中 |
| ⏸️ Queued | 排队中 |

### 测试点详情

每个测试点显示：
- **判题结果**：AC/WA/TLE/MLE/RE 等
- **运行时间**：程序执行时间（毫秒）
- **内存占用**：程序内存使用（KB）
- **错误信息**：如有错误，显示详细信息

## 6. 参加比赛

### 查看比赛列表

1. 点击导航栏「比赛」进入比赛列表
2. 查看比赛状态：
   - **未开始**：显示倒计时
   - **进行中**：可参与
   - **已结束**：可查看题目和排行榜

### 参加有密码的比赛

1. 点击进入比赛详情页
2. 如需密码，点击「输入密码」按钮
3. 输入管理员提供的比赛密码
4. 验证通过后即可查看题目

### 比赛中提交

1. 在比赛题目列表中点击题目
2. 点击「提交」按钮（比赛模式下评测模式由管理员设定，不可更改）
3. 编写代码并提交
4. 在排行榜中查看成绩

### 排行榜说明

- **ACM 模式**：按通过题数排序，题数相同则按罚时排序
- **OI 模式**：按总分排序
- **封榜**：比赛结束前一段时间可能封榜，排行榜不再更新

## 7. 个人设置

### 修改偏好设置

1. 点击右上角头像，选择「设置」
2. 可修改以下设置：
   - **默认编程语言**：提交代码时自动选择
   - **默认判题模式**：ACM 或 OI
   - **界面语言**：简体中文 / English
   - **主题**：浅色 / 深色 / 跟随系统

### 查看个人资料

1. 点击右上角头像，选择「个人资料」
2. 查看提交统计和做题记录

## 常见问题

### Q: 提交后显示「Compile Error」怎么办？
A: 检查代码语法是否正确，确保使用的语言和选择的语言一致。

### Q: 提交后显示「Time Limit Exceeded」怎么办？
A: 程序运行时间超出限制，请优化算法复杂度。

### Q: 如何查看其他人的代码？
A: 在提交列表中可以查看他人的提交记录和代码（取决于系统配置）。

### Q: 忘记密码怎么办？
A: 请联系管理员重置密码。重置后使用原用户名在注册页设置新密码。
