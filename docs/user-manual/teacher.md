# 教师手册

本手册帮助教师（PROBLEM_EDITOR 角色）使用 Epoch Judge 系统管理题目和查看学生成绩。

## 1. 权限说明

教师角色（PROBLEM_EDITOR）拥有以下权限：

- ✅ 创建和编辑题目
- ✅ 管理题目的测试用例
- ✅ 导入/导出题目
- ✅ 创建和管理比赛
- ✅ 查看所有学生的提交记录和成绩
- ✅ 发布公告
- ✅ 重判提交

> 💡 如需更高权限（如用户管理、系统配置），请联系管理员升级为 ADMIN 角色。

## 2. 创建题目

### 操作步骤

1. 登录后进入「管理后台」（导航栏右侧）
2. 切换到「题目」标签页
3. 点击「新建题目」按钮
4. 填写题目信息：
   - **标题**（必填）：题目名称
   - **可见性**：PUBLIC（公开）或 PRIVATE（私有，仅比赛可见）
   - **标签**（选填）：用于分类筛选
5. 点击「创建」完成

### 编辑题目详情

1. 在题目列表中找到目标题目，点击「编辑」
2. 编辑以下内容：
   - **题目描述**：支持 Markdown 格式
   - **难度**：0-5000 的数值
   - **时间限制**：毫秒（默认 1000ms）
   - **内存限制**：KB（默认 256MB）
   - **标签**：添加分类标签
3. 切换到「测试用例」标签页管理测试用例
4. 点击「保存」

## 3. 管理测试用例

### 添加测试用例

1. 在题目编辑页面切换到「测试用例」标签
2. 添加测试用例：
   - **输入数据**：程序的标准输入
   - **期望输出**：程序应该输出的正确答案
3. 标记样例测试点（对用户可见）

### 测试用例说明

- **样例测试点**：用户可见，用于帮助理解题目
- **隐藏测试点**：用户不可见，用于评判代码正确性
- 建议至少准备 3-5 个测试点，覆盖边界情况

## 4. 导入导出题目

### 导出题目

1. 在题目列表中找到目标题目
2. 点击「导出」按钮
3. 下载 JSON 格式的题目文件

### 导入题目

1. 在题目列表上方点击「导入题目」
2. 上传 JSON 格式的题目文件
3. 确认导入

> 💡 导入导出功能可用于在不同环境间迁移题目，或备份题目数据。

## 5. 创建比赛

### 操作步骤

1. 进入「管理后台」→「比赛」标签页
2. 点击「新建比赛」按钮
3. 填写比赛信息：

| 字段 | 说明 |
|------|------|
| 标题 | 比赛名称 |
| 说明 | 比赛描述（支持 Markdown） |
| 可见性 | PUBLIC / PRIVATE |
| 评测模式 | ACM / OI |
| 开始时间 | 比赛开始时刻 |
| 结束时间 | 比赛结束时刻 |
| 封榜 | 结束前多久封榜（可选） |
| 比赛密码 | 访问密码（可选，留空无密码） |
| 比赛题目 | 选择题目，拖拽排序（对应 A/B/C…） |

4. 点击「保存」

### 比赛时间设置

- **开始时间**：比赛何时开始，精确到分钟
- **时长**：可以选择「开始后 N 小时/分钟」或直接指定结束时刻
- **封榜**：比赛结束前多久停止更新排行榜（如最后 30 分钟）

### 比赛题目管理

- 从题目列表中选择题目添加到比赛
- 拖拽调整题目顺序（顺序对应 A、B、C… 标签）
- 可随时添加或移除题目

## 6. 管理比赛报名

### 查看报名情况

1. 在比赛列表中点击「报名/打星」按钮
2. 查看已报名的学生列表

### 管理参赛者

- 查看报名时间
- 设置打星队伍（不参与排名）
- 管理比赛密码验证

## 7. 查看成绩

### 查看提交记录

1. 进入「管理后台」→「重判」标签页
2. 可按以下维度筛选提交：
   - **按提交**：选择特定提交记录
   - **按题目**：选择特定题目的所有提交
   - **按比赛**：选择特定比赛的所有提交
3. 筛选状态：AC、WA、TLE、MLE、RE 等

### 查看比赛排行榜

1. 进入比赛详情页
2. 查看实时排行榜：
   - **ACM 模式**：按通过题数和罚时排序
   - **OI 模式**：按总分排序
3. 排行榜每 10 秒自动刷新

### 重判提交

1. 在「重判」标签页选择要重判的提交
2. 点击「预览」查看影响范围
3. 确认后点击「重判」执行

> ⚠️ 重判操作会重新评测选定的所有提交，请谨慎操作。

## 8. 发布公告

### 操作步骤

1. 进入「管理后台」→「公告」标签页
2. 点击「新建公告」
3. 填写公告信息：
   - **标题**（必填）
   - **内容**（必填）
   - **置顶**：是否置顶显示
   - **生效时间**：公告何时开始显示（可选）
   - **失效时间**：公告何时停止显示（可选）
4. 点击「确定」发布

### 公告管理

- 编辑已有公告
- 删除公告
- 设置/取消置顶
- 设置生效时间范围

## 9. 批量导入学生

### 操作步骤

1. 进入「管理后台」→「用户」标签页
2. 点击「下载模板」获取 CSV 模板
3. 按模板格式填写学生信息
4. 点击「批量导入 CSV」上传文件
5. 系统自动创建学生账号

### CSV 模板格式

```csv
username,password,email,displayName
student001,password123,student001@example.com,张三
student002,password456,student002@example.com,李四
```

> 💡 批量导入适合学期初统一创建学生账号。

## 常见问题

### Q: 如何修改已发布题目的测试用例？
A: 进入管理后台 → 题目 → 编辑 → 测试用例标签，修改后保存即可。如有需要，可对已提交的记录进行重判。

### Q: 比赛开始后还能修改题目吗？
A: 可以修改，但建议在比赛前完成所有修改，避免影响比赛进行。

### Q: 如何查看某个学生的所有提交？
A: 在重判标签页中按题目筛选，或直接在提交列表页面查看。

### Q: 如何设置私有题目？
A: 创建题目时选择「PRIVATE」可见性，私有题目仅在比赛中可见。
