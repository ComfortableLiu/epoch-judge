## ADDED Requirements

### Requirement: Student manual exists
The system SHALL provide a student manual covering registration, problem browsing, code submission, and contest participation.

#### Scenario: Student reads manual
- **WHEN** student accesses the user manual
- **THEN** system displays instructions for common student workflows

### Requirement: Teacher manual exists
The system SHALL provide a teacher manual covering problem creation, homework management, and grade tracking.

#### Scenario: Teacher reads manual
- **WHEN** teacher accesses the user manual
- **THEN** system displays instructions for teacher-specific workflows

### Requirement: Admin manual exists
The system SHALL provide an admin manual covering system configuration, user management, and contest management.

#### Scenario: Admin reads manual
- **WHEN** admin accesses the user manual
- **THEN** system displays instructions for admin-specific workflows
