## Context

管理员无法直观了解平台使用情况，缺乏数据驱动决策的基础。需要新增统计仪表盘。

## Goals / Non-Goals

**Goals:**
- 管理后台新增数据统计页面
- 展示 DAU、提交量、判题延迟等核心指标
- 展示趋势图表和热门题目

**Non-Goals:**
- 不做实时数据（定时聚合即可）
- 不做用户行为分析
- 不做数据导出

## Decisions

### 1. 数据聚合
- **选择**：定时任务每小时聚合，结果存 Redis
- **理由**：避免每次请求都做全表扫描

### 2. 图表库
- **选择**：Ant Design Charts（@ant-design/charts）
- **理由**：与项目 UI 框架一致，开箱即用

## Risks / Trade-offs

- [性能] 聚合查询可能很慢 → 使用 Redis 缓存，定时任务在低峰期执行
- [准确性] 数据有延迟（最多 1 小时）→ 可接受，管理场景不需要实时

## Migration Plan

1. API 新增 stats 模块
2. 前端管理后台新增 Dashboard 页面
3. 添加定时聚合任务
