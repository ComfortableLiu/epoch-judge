## Why

当前文档仅有面向开发者的 deploy.md，缺少面向最终用户（学生、教师、管理员）的操作手册。新用户上手成本高，管理员不知道如何配置比赛、导入题目。

## What Changes

- 编写面向用户的角色分册：学生手册、教师手册、管理员手册
- 学生手册：注册登录、浏览题目、提交代码、查看结果、参加比赛
- 教师手册：创建题目、导入题库、布置作业、查看成绩
- 管理员手册：系统配置、用户管理、判题节点管理、比赛管理
- 部署到 GitHub Pages 或项目 wiki
- 含截图和 GIF 操作演示

## Capabilities

### New Capabilities

- `user-manual`: 面向最终用户的操作手册（学生/教师/管理员）

### Modified Capabilities

（无）

## Impact

- **文档**：新增 `docs/user-manual/` 目录
- **无代码变更**
- **部署**：可选 GitHub Pages
